# DeepAudit Pro 部署指南

## 一、服务器准备

### 1.1 腾讯云服务器配置
- **推荐配置**：2核4G内存，50G硬盘
- **操作系统**：Ubuntu 20.04 LTS 或 CentOS 7.6+
- **带宽**：3M以上

### 1.2 安装宝塔面板

**Ubuntu/Debian 系统**：
```bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh
```

**CentOS 系统**：
```bash
yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh
```

安装完成后记录：
- 宝塔面板地址
- 用户名
- 密码

## 二、宝塔面板配置

### 2.1 登录宝塔面板
访问 `http://你的服务器IP:8888`，使用安装时显示的账号密码登录。

### 2.2 安装软件（在软件商店中）
- **Nginx** 1.20+
- **Python 项目管理器** 3.9+
- **PM2 管理器** 5.2+
- **MySQL** 5.7+（可选，推荐生产环境使用）

### 2.3 安全组配置
在腾讯云控制台配置安全组规则：
- 开放 80 端口（HTTP）
- 开放 443 端口（HTTPS）
- 开放 8888 端口（宝塔面板）
- 开放 22 端口（SSH）

## 三、上传项目文件

### 3.1 创建项目目录
在宝塔面板 → 文件 → 创建目录：
```
/www/wwwroot/deepaudit_pro
```

### 3.2 上传文件
方式一：使用宝塔面板上传
- 将整个项目文件夹打包成 zip
- 在宝塔面板中上传并解压

方式二：使用 Git（推荐）
```bash
cd /www/wwwroot
git clone <你的仓库地址> deepaudit_pro
```

方式三：使用 FTP/SFTP 工具
- 使用 FileZilla 等工具连接服务器
- 上传项目文件

## 四、配置环境

### 4.1 SSH 连接服务器
```bash
ssh root@你的服务器IP
cd /www/wwwroot/deepaudit_pro
```

### 4.2 配置环境变量
```bash
# 复制配置文件
cp .env.production .env

# 编辑配置文件
nano .env
```

**必须修改的配置**：
```bash
# 生成随机密钥（在本地执行）
python3 -c "import secrets; print(secrets.token_hex(32))"

# 将生成的密钥填入 .env
SECRET_KEY=生成的随机密钥

# 如果使用 MySQL
DATABASE_URL=mysql+pymysql://用户名:密码@localhost:3306/deepaudit_pro
```

### 4.3 执行部署脚本
```bash
# 添加执行权限
chmod +x deploy.sh

# 执行部署
./deploy.sh

# 创建管理员账户
./deploy.sh --create-admin
```

## 五、配置 Nginx

### 5.1 在宝塔面板创建网站
1. 点击 "网站" → "添加站点"
2. 域名：填写你的域名或服务器IP
3. 根目录：`/www/wwwroot/deepaudit_pro`
4. PHP版本：选择 "纯静态"

### 5.2 配置反向代理
1. 点击网站设置 → "反向代理"
2. 添加反向代理：
   - 代理名称：deepaudit
   - 目标URL：http://127.0.0.1:5000
   - 发送域名：$host
   - 内容替换：留空

或者直接编辑配置文件：
1. 点击网站设置 → "配置文件"
2. 将 `nginx.conf` 的内容复制进去（修改域名）
3. 保存

## 六、启动应用

### 6.1 使用 PM2 启动（推荐）

在宝塔面板 → Python项目管理器：
1. 添加项目
   - 项目路径：`/www/wwwroot/deepaudit_pro`
   - Python版本：3.9+
   - 启动文件：选择 gunicorn
   - 启动命令：`gunicorn -c gunicorn_config.py app:app`

或者使用命令行：
```bash
cd /www/wwwroot/deepaudit_pro
source venv/bin/activate
pm2 start gunicorn_config.py --name deepaudit --interpreter python3
pm2 save
pm2 startup
```

### 6.2 PM2 常用命令
```bash
pm2 list              # 查看进程列表
pm2 logs deepaudit    # 查看日志
pm2 restart deepaudit # 重启应用
pm2 stop deepaudit    # 停止应用
pm2 delete deepaudit  # 删除应用
```

## 七、配置 SSL 证书（可选）

### 7.1 申请免费证书
在宝塔面板 → 网站设置 → SSL：
1. 选择 "Let's Encrypt"
2. 输入邮箱
3. 点击申请

### 7.2 强制 HTTPS
在 SSL 设置中开启 "强制HTTPS"

## 八、数据库配置（使用 MySQL）

### 8.1 创建数据库
在宝塔面板 → 数据库：
1. 添加数据库
   - 数据库名：deepaudit_pro
   - 用户名：deepaudit_user
   - 密码：设置强密码

### 8.2 修改配置
编辑 `.env` 文件：
```bash
DATABASE_URL=mysql+pymysql://deepaudit_user:密码@localhost:3306/deepaudit_pro
```

### 8.3 迁移数据
```bash
cd /www/wwwroot/deepaudit_pro
source venv/bin/activate
python3 scripts/verify_init.py
```

## 九、测试访问

### 9.1 访问网站
浏览器打开：`http://你的域名或IP`

### 9.2 检查日志

# 应用日志
tail -f logs/gunicorn_error.log

## 十、日常维护

### 10.1 更新代码
```bash
cd /www/wwwroot/deepaudit_pro
git pull
source venv/bin/activate
pip install -r requirements.txt
pm2 restart deepaudit
```

### 10.2 备份数据库
在宝塔面板 → 计划任务：
- 添加备份数据库任务
- 执行周期：每天凌晨3点

### 10.3 监控
在宝塔面板 → 监控：
- 查看CPU、内存使用情况
- 设置告警通知

## 十一、常见问题

### 11.1 端口被占用
```bash
# 查看端口占用
netstat -tunlp | grep 5000

# 修改端口（在 .env 中）
FLASK_PORT=5001
```

### 11.2 权限问题
```bash
cd /www/wwwroot/deepaudit_pro
chmod -R 755 .
chmod -R 777 logs instance uploads exports
```

### 11.3 依赖安装失败
```bash
# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 11.4 数据库连接失败
- 检查 MySQL 是否启动
- 检查数据库用户名密码
- 检查防火墙设置

## 十二、性能优化

### 12.1 Gunicorn 优化
根据服务器配置调整 `gunicorn_config.py` 中的 workers 数量。

### 12.2 Nginx 缓存
在 Nginx 配置中启用静态文件缓存（已配置）。

### 12.3 数据库优化
- 定期清理日志表
- 添加索引
- 使用 Redis 缓存（可选）
