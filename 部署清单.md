# DeepAudit Pro 快速部署清单

## 部署前准备
- [ ] 腾讯云服务器已购买（2核4G推荐）
- [ ] 宝塔面板已安装
- [ ] 域名已备案（可选）

## 第一步：安装宝塔面板
```bash
# Ubuntu
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh

# CentOS
yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh
```

## 第二步：宝塔面板安装软件
- [ ] Nginx 1.20+
- [ ] Python 项目管理器 3.9+
- [ ] PM2 管理器
- [ ] MySQL 5.7+（可选）

## 第三步：上传项目
1. 在宝塔面板创建目录：`/www/wwwroot/deepaudit_pro`
2. 上传项目文件（zip上传或Git克隆）

## 第四步：配置环境
```bash
cd /www/wwwroot/deepaudit_pro

# 复制配置文件
cp env.production.example .env

# 生成密钥（在本地执行）
python3 -c "import secrets; print(secrets.token_hex(32))"

# 编辑 .env，填入生成的密钥
nano .env
```

## 第五步：执行部署
```bash
chmod +x deploy.sh
./deploy.sh
./deploy.sh --create-admin
```

## 第六步：配置 Nginx
在宝塔面板：
1. 网站 → 添加站点
2. 域名填写你的域名或IP
3. 网站设置 → 配置文件 → 粘贴 nginx.conf 内容（修改域名）

## 第七步：启动应用
```bash
cd /www/wwwroot/deepaudit_pro
source venv/bin/activate
pm2 start gunicorn_config.py --name deepaudit --interpreter python3
pm2 save
pm2 startup
```

## 第八步：测试访问
浏览器打开：http://你的域名或IP

## 常用命令
```bash
pm2 list              # 查看进程
pm2 logs deepaudit    # 查看日志
pm2 restart deepaudit # 重启
pm2 stop deepaudit    # 停止
```

## 问题排查
```bash
# 查看应用日志
tail -f logs/gunicorn_error.log

# 查看 Nginx 日志
tail -f /www/wwwlogs/deepaudit_error.log

# 检查端口占用
netstat -tunlp | grep 5000
```
