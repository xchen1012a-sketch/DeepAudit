## 功能特性

### ✅ 智能识别（按优先级）
1. **环境变量识别**
   - 解析 `DATABASE_URL` / `SQLALCHEMY_DATABASE_URI`
   - 自动排除 demo/clean 路径

2. **配置文件解析**
   - 读取 `core/settings.py` 默认配置
   - 识别 `sqlite:///database.db`

3. **兜底规则**
   - 扫描所有 `.db` 文件
   - 排除文件名包含 demo/clean 的库
   - 选择最近修改的数据库

### ✅ 安全保证
- 只读操作，不修改原数据库
- 自动备份已存在的导出文件
- 严格过滤 demo/clean 库
- 导出后验证数据库完整性

### ✅ 用户体验
- 双击即可运行（`scripts\export_real_db.bat`）
- 详细的识别过程输出
- 完整的 Navicat 导入步骤
- UTF-8 编码支持（解决 Windows 控制台乱码）

## 技术实现

### 核心逻辑
```python
1. 检查环境变量 (DATABASE_URL / SQLALCHEMY_DATABASE_URI)
2. 解析配置文件 (core/settings.py)
3. 扫描所有 .db 文件
4. 过滤 demo/clean 库
5. 选择最新修改的真实库
6. 复制到 exports/ 目录
7. 验证数据库完整性
8. 输出 Navicat 导入指南
```

### 依赖项
- Python 3.8+
- 标准库：os, sys, shutil, sqlite3, datetime, pathlib
- 无需额外安装依赖

### 编码处理
```python
# Windows 控制台 UTF-8 支持

```
