## 🎯 核心功能

### ✅ 已实现的功能

1. **数据库完全隔离**
   - `database.db` - 生产环境（手动启动）
   - `database_demo.db` - 演示环境（run_demo.bat）
   - `database_clean.db` - 开发环境（run_clean.bat）

2. **自动创建管理员账号**
   - 两个启动脚本都会自动创建 admin01/123456
   - 无需手动创建或复制账号
   - 密码统一为 123456

3. **自动打开浏览器**
   - Flask 启动后 3 秒自动打开浏览器
   - 访问 http://127.0.0.1:5000

4. **跳过手动确认**
   - 显示提示后 3 秒自动继续
   - 无需手动输入 Y

5. **防止误删生产库**
   - 启动脚本只操作 demo/clean 数据库
   - database.db 永不删除

## 🐛 常见问题

### Q1: 登录提示密码错误

**A**: 运行以下命令重置密码：
```cmd
python scripts\create_admin.py
```

### Q2: 浏览器没有自动打开

**A**: 手动打开浏览器访问 http://127.0.0.1:5000

### Q3: 提示找不到 werkzeug 模块

**A**: 确保虚拟环境已激活并安装依赖：
```cmd
.venv\Scripts\activate
pip install -r requirements.lock.txt
```

### Q4: database.db 被删除了

**A**: 不可能！脚本只删除 database_demo.db 和 database_clean.db，绝不删除 database.db
