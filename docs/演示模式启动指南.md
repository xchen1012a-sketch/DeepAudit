## 🐛 故障排除

### 问题 1: 脚本闪退

**解决方法**：
使用调试版本查看详细错误：
```
双击: scripts\run_demo_debug.bat
```

查看日志文件：
```
run_demo_debug.log
```

### 问题 2: Python 未安装

**错误信息**：
```
[ERROR] Python 未安装或不在 PATH 中
```

**解决方法**：
1. 安装 Python 3.8+
2. 安装时勾选 "Add Python to PATH"

### 问题 3: 依赖安装失败

**解决方法**：
手动安装依赖：
```cmd
cd C:\Users\画桦\Desktop\DeepAudit_Pro
.venv\Scripts\activate
pip install -r requirements.lock.txt
```

### 问题 4: 数据库初始化失败

**解决方法**：
1. 关闭所有占用数据库的程序
2. 删除 `database_demo.db` 后重试
3. 查看调试日志

### 问题 5: 登录密码错误

**解决方法**：
运行账号创建脚本：
```cmd
cd C:\Users\画桦\Desktop\DeepAudit_Pro
.venv\Scripts\activate
python scripts\create_admin_demo.py
```
