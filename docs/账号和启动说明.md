## 📊 数据库隔离说明

### 三个独立数据库

| 数据库文件 | 用途 | 启动方式 | 账号 |
|-----------|------|---------|------|
| `database.db` | **生产环境** | 手动启动 `flask run` | admin01/123456 |
| `database_demo.db` | **演示环境** | `scripts\run_demo.bat` | admin01/123456 |
| `database_clean.db` | **开发环境** | `scripts\run_clean.bat` | admin01/123456 |

### 数据隔离保证

✅ **三个数据库完全独立**
- 通过环境变量 `DATABASE_URL` 切换
- 互不影响，数据完全隔离

✅ **database.db 永不删除**
- 启动脚本只操作 `database_demo.db` 和 `database_clean.db`
- 生产数据绝对安全

## 🚀 三种启动方式

### 方式一：演示模式（推荐用于演示）

**启动命令**：

### 方式二：纯净模式（推荐用于开发）

## 🔧 账号管理工具

### 1. 创建/重置 admin01 账号

如果忘记密码或账号丢失，运行：

```cmd
python scripts\create_admin.py
```

**功能**：
- 检查 `database.db` 中是否有 admin01
- 如果存在：重置密码为 123456
- 如果不存在：创建新账号 admin01/123456

---

### 2. 检查所有数据库的账号

```cmd
python scripts\check_admin.py
```

**功能**：
- 检查三个数据库中的 admin01 账号
- 如果不存在，自动创建
- 显示账号详细信息
