# DeepAudit Pro 项目清理指南

## 📊 可以安全删除的文件

### 1. Python 缓存文件（__pycache__）
这些是 Python 自动生成的缓存文件，可以安全删除，会自动重新生成。

**位置**：
- `__pycache__/` （项目根目录）
- `core/__pycache__/`
- `events/__pycache__/`
- `integrations/__pycache__/`
- `migrations/__pycache__/`
- `migrations/versions/__pycache__/`
- `models/__pycache__/`
- `providers/__pycache__/`
- `routes/__pycache__/`
- `scripts/__pycache__/`
- `tasks/__pycache__/`
- `tests/__pycache__/`
- `tools/__pycache__/`

**删除命令**：
```cmd
cd C:\Users\画桦\Desktop\DeepAudit_Pro
for /d /r . %d in (__pycache__) do @if exist "%d" rd /s /q "%d"
```

## 📊 预计清理空间

## ⚠️ 注意事项

1. **不要删除 database.db**
   - 这是生产数据库，包含真实数据
   - 删除后无法恢复

2. **不要删除 .venv 文件夹**
   - 这是 Python 虚拟环境
   - 删除后需要重新安装所有依赖

3. **不要删除源代码文件**
   - 所有 .py 文件都是源代码
   - 删除后系统无法运行

4. **备份重要数据**
   - 清理前建议备份 database.db
   - 以防万一
