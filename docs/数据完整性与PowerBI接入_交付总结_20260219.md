## 二、Power BI 接入方案

### 2.1 架构设计

```
业务数据库 (database.db)
    ↓
后端聚合服务 (pbi_aggregation_service.py)
    ↓
Web API 接口 (pbi_api.py)
    ↓ CSV/JSON
Power BI Desktop/Service
```

### 2.2 核心组件

#### 1. 聚合服务层
**文件**：`services/pbi_aggregation_service.py`

**功能**：
- `aggregate_daily_metrics()` - 按天聚合指标
- `aggregate_action_metrics()` - 按动作聚合指标
- `aggregate_risk_metrics()` - 按风险状态聚合指标
- `aggregate_department_metrics()` - 按部门聚合指标
- `aggregate_dashboard_summary()` - 综合仪表板数据

**特性**：
- ✅ 使用 SQL 聚合查询，性能优化
- ✅ 集成缓存装饰器，避免重复计算
- ✅ 支持日期范围筛选

#### 2. API 路由层
**文件**：`routes/pbi_api.py`

**接口列表**：
- `GET /api/pbi/metrics/daily` - 每日指标
- `GET /api/pbi/metrics/actions` - 动作指标
- `GET /api/pbi/metrics/risks` - 风险指标
- `GET /api/pbi/metrics/departments` - 部门指标
- `GET /api/pbi/dashboard` - 综合仪表板
- `GET /api/pbi/health` - 健康检查

**特性**：
- ✅ 支持 CSV/JSON 双格式输出
- ✅ 支持日期范围参数（start_date, end_date）
- ✅ 默认查询最近 90 天数据
- ✅ API Key 认证（可选）

#### 3. 缓存工具
**文件**：`utils/pbi_cache.py`

**功能**：
- 简单的内存缓存实现
- TTL 默认 15 分钟
- 支持装饰器模式
- 自动过期清理

**特性**：
- ✅ 避免重复计算，提升响应速度
- ✅ 缓存键基于参数自动生成
- ✅ 支持手动清空缓存

### 2.3 API 使用示例

#### 示例 1：获取每日指标（CSV 格式）
```
http://localhost:5000/api/pbi/metrics/daily?start_date=2026-01-01&end_date=2026-02-19&format=csv
```

**返回字段**：
- date, invoice_count, invoice_amount, risk_event_count, risk_case_count
- bank_txn_count, bank_txn_amount, high_risk_count, medium_risk_count, low_risk_count

#### 示例 2：获取综合仪表板（JSON 格式）
```
http://localhost:5000/api/pbi/dashboard?start_date=2026-01-01&end_date=2026-02-19
```

**返回数据**：
```json
{
  "ok": true,
  "data": {
    "period": {"start_date": "2026-01-01", "end_date": "2026-02-19"},
    "invoices": {
      "total_count": 80,
      "total_amount": 405562.55,
      "high_risk_count": 6,
      "approved_count": 24
    },
    "risk_events": {"total_count": 40},
    "risk_cases": {"total_count": 20, "close_rate": 35.0},
    "bank_transactions": {"total_count": 150, "match_rate": 50.0}
  }
}
```

### 2.4 Power BI 配置步骤

1. **启动应用**：`python app.py`
2. **测试 API**：访问 `http://localhost:5000/api/pbi/health`
3. **Power BI Desktop**：
   - 获取数据 → Web
   - 输入 URL：`http://localhost:5000/api/pbi/metrics/daily?format=csv`
   - 加载数据
4. **创建可视化**：
   - 折线图：每日趋势
   - 饼图：风险分布
   - 柱状图：部门对比
   - 热力图：动作分布

### 2.5 安全配置

#### API Key 认证（可选）

1. 在 `.env` 文件中设置：
```env
PBI_API_KEY=your_secure_api_key_here
```

2. 在 URL 中添加 API Key：
```
http://localhost:5000/api/pbi/metrics/daily?format=csv&api_key=your_secure_api_key_here
```

3. 或在 HTTP Header 中添加：
```
X-API-Key: your_secure_api_key_here
```
