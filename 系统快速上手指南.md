## 💡 系统是什么？

**DeepAudit Pro** = 智能费用审计系统

帮你实现：**发票上传 → 自动识别 → 风险评估 → 在线审批 → 数据归档**

---

## 📌 核心功能（5个）

### 1️⃣ 费用报销受理中心
**上传发票，自动识别**
- 拖拽上传发票图片/PDF
- AI自动识别金额、日期、商户
- 一键提交审批

### 2️⃣ 审批管理工作台
**在线审批，高效便捷**
- 查看待审批单据
- 查看风险评估结果
- 通过/退回/转派

### 3️⃣ 风控中心
**智能风控，实时预警**
- 自动评估风险等级（高/中/低）
- 异常交易预警
- 风险案件处理

### 4️⃣ 凭证台账
**数据归档，一键导出**
- 查看所有已入账单据
- 按日期/部门/金额筛选
- 导出Excel报表

### 5️⃣ 管理总览
**数据看板，一目了然**
- 今日提交数、待审批数
- 高风险案件数、通过率
- 趋势图表分析

---

## 🎯 典型使用流程

```
员工提交发票
    ↓
```

## ⚡ 3分钟上手操作

### 场景1：我要报销一张发票
1. 点击 **费用报销受理中心**
2. 拖拽上传发票图片
3. 确认识别信息 → 点击 **上传并校验**
4. 完成！等待审批通知

### 场景2：我要审批单据
1. 点击 **审批管理工作台**
2. 点击待审批单据查看详情
3. 查看发票信息和风险评估
4. 选择 **通过/退回/转派** → 填写意见 → 提交

### 场景3：我要查看数据
1. 点击 **凭证台账**
2. 设置筛选条件（日期/部门/金额）
3. 点击 **导出报表** 下载Excel

## 🔧 常见问题（3个必看）

**Q1：上传后找不到单据？**
→ 在"我的单据"列表中查看，或检查是否上传成功

**Q2：单据显示"待补录"？**
→ 点击单据补充金额和日期，然后点击"入账并进入审批"

**Q3：审批时提示"无权访问"？**
→ 不能审批自己提交的单据，或该单据不在你的权限范围
